# prova_02_joao_marcos_melchioretto_eletronica
Calculadora Elétrica – Prova 02

Este projeto consiste em uma aplicação web simples desenvolvida em HTML e JavaScript para auxiliar em cálculos básicos de eletrônica. Ele reúne diversas ferramentas úteis para estudantes e praticantes da área.

Funcionalidades

A aplicação possui os seguintes módulos:

    1. Cálculo de Potência Mensal
        Calcula o consumo energético (kWh/mês) de um eletrodoméstico.
        Estima o custo com base no valor do kWh.
        Entrada de potência, tempo de uso diário e tarifa.
        
    2. Conversor de Unidades Elétricas
        Converte valores entre diferentes escalas (T, G, M, k, m, µ, n, p).
        Suporta unidades como:
        Ampère (A)
        Ohm (Ω)
        Volt (V)
        Watt (W)
        Joule (J)
        Coulomb (C)

    3. Reconhecimento de Resistores
        Calcula o valor de resistores a partir do código de cores.
        Suporte para resistores de 4 e 5 faixas.
        Exibe:
        Resistência
        Tolerância

    4. Lei de Ohm
        Calcula uma grandeza elétrica desconhecida:
        Tensão (U)
        Corrente (I)
        Resistência (R)
        Baseado na fórmula: U = R × I

    5. Associação de Resistores
        Calcula resistência equivalente em:
        Série
        Paralelo

    6. Associação de Geradores
        Calcula:
        Tensão equivalente (Ueq)
        Corrente equivalente (Ieq)
        Considera associação em série ou paralelo.

Como Executar
    Baixe ou clone o repositório.
    Abra o arquivo index.html em qualquer navegador.
    Navegue pelas funcionalidades utilizando os botões.

Observações
    O projeto é totalmente client-side (não necessita servidor).
    Ideal para fins educacionais.
    Interface simples e direta para facilitar o uso.
